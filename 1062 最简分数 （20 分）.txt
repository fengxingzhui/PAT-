1062 最简分数 （20 分）
一个分数一般写成两个整数相除的形式：N/M，其中 M 不为0。最简分数是指分子和分母没有公约数的分数表示形式。

现给定两个不相等的正分数 N1/M​1和 N2/M2，要求你按从小到大的顺序列出它们之间分母为 K 的最简分数。

输入格式：
输入在一行中按 N/M 的格式给出两个正分数，随后是一个正整数分母 K，其间以空格分隔。题目保证给出的所有整数都不超过 1000。

输出格式：
在一行中按 N/M 的格式列出两个给定分数之间分母为 K 的所有最简分数，按从小到大的顺序，其间以 1 个空格分隔。行首尾不得有多余空格。题目保证至少有 1 个输出。

输入样例：
7/18 13/20 12
输出样例：
5/12 7/12


注意点： 有一个坑点是N1/M1 不一定小于 N2/M2；


#include<cstdio>
using namespace std;

int gcd(int a,int b)
{
	return !b?a:gcd(b,a%b);
}
int main()
{
	int N1,M1,N2,M2,K,tmp;
	scanf("%d/%d %d/%d %d",&N1,&M1,&N2,&M2,&K);
	if(N1*M2>N2*M1) 
	{
		tmp=N2;
		N2=N1;
		N1=tmp;
		tmp=M2;
		M2=M1;
		M1=tmp;
	}
	int a[1000]={},cnt=0;
	for(int i=1;i<K;i++)
	{
		if(i*M1>K*N1&&i*M2<K*N2&&gcd(i,K)==1)
		{
			if(cnt) printf(" ");
			printf("%d/%d",i,K);
			cnt++;
		}
	}
	
}
